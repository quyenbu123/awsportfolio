> **Note:** April 30th (Liberation Day) and May 1st (International Workers' Day) are public holidays — this week contains only 3 working days.

### Week Topic

AWS technical documentation research, Console familiarization, and EC2 resource management.

### Week Objectives

* Deepen understanding of AWS service mechanisms and use cases.
* Master AWS Console navigation and EC2 computing resources.
* Analyze secure integration from EC2 to other AWS services without static access keys.

### Expected Outcomes

* Understand how IAM Roles and Instance Profiles secure applications running on EC2.
* Gain confidence in navigating and managing resources via the AWS Console.
* Spin up, configure, and connect to both Linux (via SSH/Session Manager) and Windows (via RDP) EC2 instances.
* Master EC2 deployment terms: AMI, Instance Types, Security Groups, and Key Pairs.
