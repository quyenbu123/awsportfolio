### Week Topic

Internship kick-off, AWS corporate overview, and core cloud services setup.

### Week Objectives

* Participate in the internship kick-off meeting and align on the First Cloud AI Journey roadmap.
* Understand AWS Vietnam operations and developer standards.
* Study AWS Core services (IAM & VPC) and create basic secure resources.

### Daily Work Log

| Date | Day | Work Items | Lab / Project |
|---|---|---|---|
| 20/04/2026 | Monday | Participate in the internship kick-off, aligning on rules, expectations, and self-study roadmap. | Study Roadmap |
| 21/04/2026 | Tuesday | Get an overview of AWS cloud platform. Create a personal AWS account and configure MFA security on root. Practice Lab 000001. | [Lab 000001 - Create new AWS Account](https://000001.awsstudygroup.com)|
| 22/04/2026 | Wednesday | Study IAM fundamentals: users, groups, and policies. Apply the Least Privilege principle. Practice Lab 000002 (Sections 1.1-2.3). | [Lab 000002 - Access Management with AWS IAM](https://000002.awsstudygroup.com)|
| 23/04/2026 | Thursday | Distinguish IAM Users from IAM Roles. Practice the Console Switch Role workflow. Continue Lab 000002 (Sections 3.1-5). | [Lab 000002 - Access Management with AWS IAM](https://000002.awsstudygroup.com)|
| 24/04/2026 | Friday | Study networking basics with Amazon VPC, custom subnets, and routing table setups. Practice Lab 000003. | [Lab 000003 - Deploy network infrastructure with Amazon VPC](https://000003.awsstudygroup.com)|

### Expected Outcomes

* Successfully register AWS account and enable MFA protection for root user.
* Understand AWS Budgets setup to monitor usage and stay within Free Tier limits.
* Define IAM users, groups, and policies applying Least Privilege best practices.
* Provision a custom VPC with Public/Private subnets, Route Tables, and Internet Gateways.
