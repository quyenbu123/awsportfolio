Events Participated

### [Event 1 - Prompt Engineering, AI Mindmap & BMAD Method](4.1-Event1/)
This session had three main parts: how to write better prompts so LLMs give more useful answers, a mindmap on effective AI interaction combined with AWS architecture, and the BMAD method—an open-source framework that uses AI to guide structured software development.

### [Event 2 - AWS Vietnam Community Day 2026](4.2-Event2/)
This was a major community event with multiple speakers sharing real-world experiences on AI and Cloud. Topics included context engineering (Build Second Brain), CloudFront optimization, multi-agent credit scoring, and hackathon experiences.

### [Event 3 - AWS Certification Quiz](4.3-Event3/)
This event was a fun and competitive AWS Certification Quiz among interns, organized into 8 teams competing across 10 question sets. It served as a great opportunity to consolidate AWS service knowledge and learn from detailed judge explanations.

### [Event 4 - FCAJ Community Day June 2026](4.4-Event4/)
This event focused on practical solutions for integrating AI into enterprise-scale system operations. Key topics included the Agentic platform for Cloud operations, building Voice AI Assistants, DevOps AI Agent on AWS, automating HR processes with Amazon Q, and establishing Private Security models.
