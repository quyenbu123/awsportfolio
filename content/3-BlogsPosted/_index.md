This section lists and introduces the blogs posted to the [AWS Study Group](https://www.facebook.com/groups/awsstudygroupfcj):

### [Blog 1 - BUILDING RELIABLE MULTI-STEP WORKFLOWS WITH AWS LAMBDA DURABLE FUNCTIONS](3.1-Blog1/)
This blog explores AWS Lambda Durable Functions, a newly released feature that allows developers to write stateful, multi-step workflows sequentially inside a single Lambda function with built-in checkpointing, automatic retries, and scale-to-zero compute pausing for up to a year.

### [Blog 2 - NEXT-GEN OPENSEARCH SERVERLESS: VECTOR SEARCH SCALE-TO-ZERO FOR AI AGENTS](3.2-Blog2/)
This blog introduces the next generation of Amazon OpenSearch Serverless (NextGen), which completely decouples compute and storage to achieve true scale-to-zero compute scaling, 20x faster capacity adjustments, and up to 60% infrastructure cost savings for AI agent applications.

### [Blog 3 - BUILDING MEMORY-INTENSIVE APPS WITH AWS LAMBDA MANAGED INSTANCES](3.3-Blog3/)
This blog highlights AWS Lambda Managed Instances, a solution enabling Serverless execution of memory-intensive workloads up to 32 GB RAM on managed EC2 instances—ideal for in-memory analytics, ML inference, and semantic search without server overhead.
